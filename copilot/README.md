# GitHub Copilot Resources

This directory contains specialized resources for GitHub Copilot to enhance B&R Industrial Automation development workflows.

## 📁 Directory Structure

- **[agents/](./agents/)** - Custom GitHub Copilot agents specialized for B&R development
- **[collections/](./collections/)** - Curated collections of related Copilot resources
- **[instructions/](./instructions/)** - Instruction files to guide Copilot's behavior and context
- **[prompts/](./prompts/)** - Reusable prompt templates for common tasks
- **[mcp/](./mcp/)** - Model Context Protocol (MCP) servers for extended functionality
- **[scripts/](./scripts/)** - Utility scripts for managing Copilot resources
- **[docs/](./docs/)** - Documentation for all Copilot resources

## 🚀 Getting Started

1. Browse the [docs/](./docs/) folder for detailed guides on each resource type
2. Install custom agents directly into VS Code
3. Configure MCP servers for enhanced capabilities
4. Apply instruction files to your projects
