## 📖 How to Use These Guidelines

### For Developers

1. Review the relevant guidelines before starting a new project
2. Reference them during code reviews
3. Use as a basis for project-specific coding standards

### With GitHub Copilot

These guidelines are automatically referenced by:
- Custom instruction files in `../copilot/instructions/`
- Custom agents in `../copilot/agents/`
- Project-specific collections

GitHub Copilot will use these guidelines to generate code that follows B&R best practices.
